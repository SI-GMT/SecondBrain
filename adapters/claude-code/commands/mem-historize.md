Move a finished project into the archived zone (10-episodes/archived/{slug}/) so it stays in the vault for long-term reference but is excluded by default from the access skills (mem-recall, mem-list, mem-search, mem-digest).

Required: --slug {project-slug}. Options: --revive (reverse the archive), --apply (without it, dry-run only), --no-confirm, --json.
