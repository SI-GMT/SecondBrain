Turn a week of session archives into a worklog — daily hours table prorated over a fixed daily amplitude, plus an activity report and next-week objectives.

Optional first argument: week_of (any YYYY-MM-DD inside the target week, default = this week). Options: --amplitude H (worked hours per day, default 7), --weekend (include Sat/Sun). Read-only, uses the optional vault template 99-meta/worklog-template.md.
