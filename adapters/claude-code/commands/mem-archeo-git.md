Phase 3 of the triphasic archeo: reconstruct the Git history of an existing repo as dated archives, enriched by the topology and stack from Phases 0/1/2.

Optional: path to the Git repo (default: CWD). Options: --level tags|releases|merges|commits, --project {slug}, --since YYYY-MM-DD, --until YYYY-MM-DD, --window day|week|month, --depth N, --rescan, --dry-run, --no-confirm. Branch-first: --branch-first {branch}, --branch-base {ref}, --by-author|--by-merge|--by-window.
